HTML Converter App (Markdown to HTML)

A web application that converts basic Markdown syntax into HTML using JavaScript. Users can input Markdown text, and the app dynamically generates the corresponding HTML output.

Project Overview

This project demonstrates frontend web development and JavaScript DOM manipulation. It allows users to write content in Markdown and instantly see the HTML result, helping developers and content creators preview HTML formatting efficiently.

Features

Convert basic Markdown constructs:

Headings (#, ##, ###)

Bold (**text**) and italic (*text*)

Lists (ordered and unordered)

Links ([text](url))

Real-time conversion as the user types

Responsive and clean UI

User-friendly and interactive interface

Technologies Used

HTML5 – Layout and structure

CSS3 – Styling and responsive design

JavaScript – Markdown parsing and HTML generation
